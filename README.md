# Installation
## Requirements
- FiveM

## Via Git ( recommended )
From your resouces directory:
```
git clone https://github.com/Lachee/fivem-busdriver.git "[lachee]/lachee-boomgate"
```

Then in you server cfg:
- `ensure lachee-boomgate`
